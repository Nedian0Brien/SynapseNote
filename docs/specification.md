# 시냅스노트 컨셉 기획안

## Background

#### 해결하고자 하는 문제

* 노트 앱은 저장에는 강하지만, 대화형 활용과 지식 회수 경험이 약함

* 챗봇은 생성에는 강하지만, 대화 중 나온 지식이 장기 자산으로 축적되지 않음

* 사용자는 점점 더 많은 정보를 AI와 대화로 얻지만, 그 내용이 휘발되고, 구조화되지 않으며, 이후 검색·학습·재사용이 어렵다

## Objective

AI 에이전트 기반의 지식 베이스

1. AI와 대화하면서 지식을 축적하고
2. 그 지식을 다시 AI가 검색, 조합, 설명하고
3. 사용자가 AI에게 주입되는 컨텍스트를 직접 통제하며
4. 장기적으로는 사용자의 사고 체계를 보조하는 세컨드 브레인

노트 앱 / AI 채팅 인터페이스 / 에이전트 오케스트레이션 기반 개인 지식 운영체제

> **“대화가 끝나면 사라지는 AI 경험을, 누적되는 개인 지식 자산으로 전환한다.”**

## Benchmark

* Obsidian / Notion 같은 지식 관리 툴

* ChatGPT / Claude 같은 대화형 AI

* NotebookLM / Perplexity 등의 RAG 기반 AI

* 개인 Second Brain 워크플로우

## Stack

* 백엔드

  * CouchDB 기반의 실시간 자동 동기화(id/pw 방식)

  * FastAPI

  * MCP Server 및 Skills를 통한 외부 연결(ChatGPT, Claude, Gemini 웹에서 연동해 사용 가능하게 할 것)

* 데이터/AI

  * LLM : Claude code(1순위), Codex(2순위), Gemini(3순위) - 골라서 사용할 수 있게 하기

  * 에이전트가 검색에 사용할 수 있는 RAG 파이프라인: Llama-Index, Qdrant

* 프론트엔드

  * Next.js / React(사용자가 주입되는 컨텍스트를 시각적인 노드나 토글 형태로 직관적으로 제어할 수 있는 반응형 웹 UI 구축

## Features

1. 에이전트와의 채팅으로 정보를 추가하고, 채팅으로 정보를 꺼내오는 에이전틱 지식 베이스
2. ChatGPT 등에서 대화하면서 얻은 지식을 실시간으로 저장할 수 있게 하기(외부 챗봇 채팅 서비스들과의 연동성)
3. 에이전트와의 채팅 과정에서 사용자가 에이전트에 주입되는 컨텍스트를 직관적으로 매니징 가능한 유저 인터페이스
4. 지식 베이스에 저장된 지식을 학습하기 위한 학습 모드(사용자의 뇌와 세컨드 브레인 역할을 하는 지식 베이스를 동기화) ex: 소크라테스식 훈련 학습 프롬프트

## Value Proposition

시냅스노트가 제공해야 하는 4가지 가치

### 1) Capture

AI와 대화하면서 나온 지식을 실시간으로 저장

* “방금 대화한 논문 요약 저장해줘”

* “이 아이디어를 프로젝트 X 지식으로 넣어줘”

* “이 답변을 내 영어 표현 노트로 축적해줘”

### 2) Recall

저장된 지식을 자연어 대화로 다시 꺼냄

* “내가 지난달 정리한 RAG 평가 기준 다시 보여줘”

* “내 군 복무 중 진행한 AI 프로젝트 중 벡터 검색 관련 내용만 모아줘”

### 3) Context Control

에이전트가 무엇을 참고하는지 사용자가 직관적으로 관리

* 현재 대화에 주입된 문서 / 메모 / 프로젝트 / 사람 / 기간 필터를 눈으로 확인

* “이 프로젝트 컨텍스트는 빼고 대답해”

* “영어공부 노트만 참조하게 해”

### 4) Learn

저장된 지식을 학습용으로 재구성

* 소크라테스식 질의

* 간격 반복

* 개념 연결 퀴즈

* ex) “내 지식 베이스 기준으로 약한 부분만 물어봐”
