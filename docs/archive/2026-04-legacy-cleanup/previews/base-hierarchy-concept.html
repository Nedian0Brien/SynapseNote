Base 개념 도입 후, 사이드바는 다음 계층을 표현해야 합니다:

전체 그래프 진입점 → 각 Base 섹션 → 소속 폴더 → 문서 순으로 트리 표시

Base는 사용자가 직접 생성·편집. 폴더를 드래그하거나 우클릭 메뉴로 Base에 추가/제거
