# 2026-04 Legacy Cleanup Archive

이 디렉터리는 2026-04 프로젝트 전반 레거시 정리 과정에서 active 경로에서 분리한 자산을
보존하기 위한 archive 공간이다.

## 범주

- `previews/`: 현재 기준 문서에서 내려온 과거 프리뷰 HTML
- `design-stitch/`: stitch 기반 디자인 실험 산출물
- `progress/`: 이전 단계 진행 로그
- `plans/`: 현재 구조와 어긋나는 과거 계획 문서
- `flask-runtime/`: FastAPI 런타임으로 대체되며 active 경로에서 내려온 Flask 기반 자산

## 분류 규칙

- `active`: 현재 실행되거나 직접 수정되는 경로
- `reference`: 현재 리뉴얼 기준 문서
- `archive`: 실행 경로에서는 제외되지만 보존이 필요한 자산

## 주의

이 archive는 영구 보존을 보장하지 않는다. 현재 목적은 삭제 전 분리 보관이며, 후속 정리에서
검토 후 추가 삭제 또는 압축 보관이 이루어질 수 있다.
